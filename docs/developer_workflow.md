# Developer Workflow

## Validate

```bash
make compile
make test
```

## Generate Simulation Report

```bash
make report
```

## Run End-To-End Offline Demo

```bash
make offline-demo
```

## Hardware Adapter Dry Run

```bash
make adapter-dry-run
```

## Optional Runtime Dependencies

Install Python dependencies for Streamlit, MQTT, plotting, and ML features:

```bash
python3 -m pip install -r requirements.txt
```
